Morse CSS Multiple Patterns Example

A demonstration of using multiple Morse patterns on a single element

Single Pattern Examples (For Comparison)

{BOLD} This paragraph is bold.

{RED} This paragraph is red.

{UNDERLINE} This paragraph is underlined.

{LARGE} This paragraph is larger.

{CENTER} This paragraph is centered.

Multiple Patterns Examples

{BOLD+RED} This paragraph is both bold and red.

{BOLD+UNDERLINE} This paragraph is both bold and underlined.

{RED+LARGE} This paragraph is both red and larger.

{BOLD+CENTER} This paragraph is both bold and centered.

{BOLD+RED+UNDERLINE} This paragraph is bold, red, and underlined.

{LARGE+CENTER+BOLD} This paragraph is larger, centered, and bold.
